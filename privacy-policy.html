Politique de Confidentialité - Eboo Kids

Dernière mise à jour : 6 janvier 2025

En bref : Eboo Kids ne collecte, ne stocke et ne partage aucune donnée personnelle en dehors de votre appareil. Toutes les informations restent strictement locales sur votre iPhone ou iPad.

1. Introduction

Eboo Kids est une application d'histoires interactives pour enfants de 3 à 8 ans. La confidentialité et la sécurité des données de nos jeunes utilisateurs sont notre priorité absolue.

2. Données collectées

L'application demande uniquement :

  • Prénom de l'enfant : utilisé pour personnaliser les histoires
  • Genre de l'enfant : utilisé pour adapter la narration (pronoms, accords)

3. Stockage des données

Toutes les données sont stockées localement sur votre appareil uniquement.

  • Utilisation d'AsyncStorage (stockage local iOS)
  • Aucune transmission à des serveurs externes
  • Aucune synchronisation cloud
  • Les données sont supprimées si vous désinstallez l'application

4. Utilisation des données

Les données collectées sont utilisées exclusivement pour :

  • Personnaliser le contenu des histoires avec le prénom de l'enfant
  • Adapter le genre grammatical dans les textes (il/elle, un/une, etc.)

5. Partage des données

Eboo Kids ne partage aucune donnée avec des tiers.

  • Pas de tracking publicitaire
  • Pas d'analytics
  • Pas de transmission à des partenaires
  • Pas de vente de données

6. Contenu de l'application

Les histoires et images sont chargées depuis notre serveur Supabase hébergé en Europe. Ces requêtes ne contiennent aucune information personnelle identifiable.

7. Sécurité

  • Connexions sécurisées HTTPS pour le chargement des histoires
  • Pas de création de compte utilisateur
  • Pas de mot de passe
  • Pas d'authentification requise

8. Publicité et contenu tiers

Eboo Kids ne contient :

  • Aucune publicité
  • Aucun contenu tiers
  • Aucun lien externe
  • Aucun tracker

9. Droits des parents

Vous pouvez à tout moment :

  • Modifier le profil : via les paramètres de l'application
  • Supprimer les données : en désinstallant l'application

10. Conformité RGPD et COPPA

Eboo Kids est conforme aux réglementations de protection des données :

  • RGPD (Europe) : Stockage local uniquement, aucune transmission de données
  • COPPA (États-Unis) : Aucune collecte de données personnelles en ligne

11. Modifications de cette politique

Nous nous réservons le droit de modifier cette politique de confidentialité. Toute modification sera publiée sur cette page avec une nouvelle date de mise à jour.

12. Contact

Pour toute question concernant cette politique de confidentialité, vous pouvez nous contacter :

  • Email : mnowebservice@gmail.com
  • Nom de l'application : Eboo Kids
  • Éditeur : Techlead Development

Last updated: January 6, 2025
